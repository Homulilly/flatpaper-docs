# 404 错误页

在站点根目录的 `source` 文件夹下新建 `404.md`，并指定 `layout: 404` 即可由主题接管 404 页面展示：

```yaml
---
title: Page not found
layout: 404
permalink: /404.html
comments: false
---
```

::: warning 注意
在使用 `hexo server` 本地预览时，随便输入一个不存在的地址并不会触发 404 页面（这是 Hexo 的默认行为）。如果需要预览该页面的实际效果，请在本地直接访问 `/404.html` 路径。部署到线上环境后，会自动由 Nginx 或 GitHub Pages 的 404 规则接管。
:::
